# 🎾 PADEX

**PADEX** es una aplicación web que permite gestionar y reservar pistas de pádel de forma sencilla.

Los usuarios pueden consultar la disponibilidad de pistas y realizar reservas desde una interfaz web moderna y responsive.

Este proyecto forma parte de mi aprendizaje en **Desarrollo de Aplicaciones Web (DAW)**.

---

## 🚀 Funcionalidades

- 📅 Consulta de disponibilidad de pistas
- 🎾 Reserva de pistas por fecha y hora
- 📱 Diseño responsive para móvil y ordenador
- ⚡ Interfaz rápida y sencilla
- 🖥 Aplicación web desarrollada con React

## 🎯 Objetivo del proyecto

El objetivo de PADEX es desarrollar una aplicación web que permita gestionar reservas de pistas de pádel de forma sencilla, aplicando conceptos de:
• desarrollo frontend moderno
• organización de proyectos con React
• diseño responsive
• creación de aplicaciones web funcionales
