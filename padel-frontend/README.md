# 🎾 PADEX – Padel Court Booking App

Aplicación web para la **gestión y reserva de pistas de pádel**.  
PADEX permite a los usuarios consultar disponibilidad de pistas y realizar reservas de forma rápida desde una interfaz web sencilla y moderna.

Este proyecto forma parte de mi **proyecto de desarrollo en DAW (Desarrollo de Aplicaciones Web)**.

---

## 🚀 Funcionalidades

- 📅 Consulta de disponibilidad de pistas
- 📝 Sistema de reservas por fecha y hora
- 🎾 Gestión de pistas de pádel
- 📱 Diseño responsive (móvil y escritorio)
- ⚡ Interfaz moderna y rápida
